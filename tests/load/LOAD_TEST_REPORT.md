# Отчет по нагрузочным тестам

## Скрипт

Я написал скрипт, который помог мне получить HTML-отчеты JMeter по двум экспериментам: с включенным и с выключенным флажком `cache_enabled` в Scrapper.

## Конфигурация

| Параметр | Значение |
|---|---:|
| Количество пользователей | 1000 |
| Подписок на пользователя | 100 |
| Доля GET `/links` запросов | 99% |
| Доля POST `/links` запросов | 1% |
| Количество потоков | 12 |
| Ramp-up duration | 60 сек |
| Stage duration | 5 мин |

## Метрики

| Experiment | RPS | Count | Avg (ms) | P50 (ms) | P99 (ms) | 200 responses | 500/502/504 responses |
|---|---:|---:|---:|---:|---:|---:|---:|
| no cache | 100,07 | 29 797 | 9,49 | 8 | 14 | 29 797 | 0 |
| cache | 108,07 | 32 394 | 1,42 | 1 | 7,99 | 32 394 | 0 |

## Заключение

Включение кеша дало заметное улучшение производительности. При этом в обоих экспериментах зафиксировано 0 ответов с ошибочными status code (500/502/504), что указывает на стабильную работу сервиса под данной нагрузкой.
