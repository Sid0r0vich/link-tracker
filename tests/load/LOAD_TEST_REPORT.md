# Отчет по нагрузочным тестам

## Скрипт

Я написал скрипт, который помог мне получить html отчеты по двум экспериментам: с включенным и с выключенным флажком cache_enabled в scrapper

## Конфигурация

|--------------------------|--------|
| Количество пользователей | 1000   |
| Подписок на пользователя | 100    |
| GET /links запросы       | 99%    | 
| POST /links запросы      | 1%     |
| Количество потоков       | 12     |
| Ramp-up duration         | 60 сек |
