# LinkTracker

**LinkTracker** – Telegram-бот, который отслеживает изменения на веб-страницах и оперативно информирует пользователя о них.


Это шаблон проекта, который вам необходимо взять за основу для разработки своей системы.


В данном файле должна находиться инструкция для ассистента по запуску и настройке бота.
Полезную для разработки проекта информацию вы можете найти в файле [HELP.md.](./HELP.md)

**Инструкция**
1. Создать .env файл по образцу .env.example
2. Создать config.yaml файл по образцу config.yaml.example
3. Запустить:
```bash
docker compose up -d --build
```
4. Тесты:
```bash
go test -v ./...
```
5. Для наглядности и тестирования по ссылкам сразу приходят обновления, будто мы подписались еще давно (закомментить строчку, если надо)
6. Фильтры не хранятся и не используются, потому что на них не было никаких конкретных заданий
